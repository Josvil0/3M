\doxysubsection{Detailed Description}
Adapter para el Recycler\+View que gestiona la visualización de una lista de películas. Esta clase es responsable de crear las vistas para cada elemento en la lista, enlazarlas con los datos de las películas y manejar los clics en los ítems.

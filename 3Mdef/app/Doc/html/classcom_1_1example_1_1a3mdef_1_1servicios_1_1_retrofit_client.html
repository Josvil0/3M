Detailed Description

Cliente Retrofit para interactuar con múltiples APIs (Google Books, TMDb y Spotify). Esta clase proporciona las instancias necesarias de Retrofit para realizar solicitudes HTTP a las APIs de Google Books, TMDb y Spotify.
